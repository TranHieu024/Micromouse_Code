Demo Tính Năng Chat

Giao diện mô phỏng quá trình khách hàng chat với Admin trên website.

Luồng hoạt động thực tế sau này:
1. Khách hàng gõ tin nhắn vào khung chat.
2. Code gửi tin nhắn qua Webhook/Resend API để đẩy thẳng vào E-mail của Admin.
3. Admin mở E-mail và bấm nút "Reply" (Trả lời) ngay trong hòm thư.
4. Hệ thống nhận được email reply, đẩy ngược tin nhắn đó về lại khung chat này cho khách hàng.

⬇️ Hãy thử bấm vào icon chat góc phải dưới để trải nghiệm!
