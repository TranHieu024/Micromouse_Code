Đăng Nhanh Hàng Loạt Tranh (POS Mode)

Quy trình đăng nhanh siêu tốc kết hợp Excel + Ảnh chụp:
Bước 1: Nhấp "Tải Ảnh Hàng Loạt" để chọn & tải lên toàn bộ ảnh chụp sản phẩm từ điện thoại vào thư viện.
Bước 2: Nhấp "Nhập File CSV từ Excel" để tải lên file Excel chứa thông tin Tên, Giá, Danh mục (không cần điền cột ảnh).
Bước 3: Nhấp nút "Chọn ảnh" ở cột đầu tiên của mỗi dòng để gán các ảnh tương ứng từ thư viện cho tranh đó.
Bước 4: Bấm "Đăng Tất Cả Lên Website" để lưu toàn bộ sản phẩm.

📷 BƯỚC 1: TẢI ẢNH HÀNG LOẠT

Tải toàn bộ ảnh sản phẩm của bạn lên thư viện ảnh tạm thời.

📊 BƯỚC 2: NHẬP FILE EXCEL / CSV

Tải file danh sách chữ (Tên, Giá, Danh mục...) soạn từ Excel.

🖼️ Thư viện ảnh tạm thời (Bấm nút "Chọn ảnh" ở bảng bên dưới để gán):
Chưa có ảnh nào được tải lên.
Hình Ảnh Tên Sản Phẩm * Danh Mục Giá bán (k) Giá gốc (k) Mô Tả & Ý Nghĩa (Tự động) Xóa
